Upmind — Complete App Reference

Comprehensive document covering all app features, 42 cognitive training exercises across 7 domains, complete user flows for every screen, and full architecture. Built with Expo SDK 51 + React Native 0.74 + Reanimated 3 + Expo Router.

Overview

Premium executive-function cognitive training for high-performers
Product
Upmind is a premium cognitive training app for founders, surgeons, pilots, and other high-performers. 10-minute daily workouts using validated neuropsychological tests — not gamification.
Tech Stack
Expo SDK 51 · React Native 0.74 · Reanimated 3 · TypeScript · Expo Router · Zustand · Maestro (E2E)
Theme
Deep navy (#0A0F1C) background, teal (#14B8A6) accent, calm Headspace-inspired motion (cubic-bezier 0.22/1/0.36/1), adaptive dark mode only.
Status
Onboarding (Welcome + ValueSlides) built. Rest is stubbed (PlaceholderScreen). 42-game catalog designed but not implemented as RN components.

Codebase structure, routing, state management

Directory Structure

  • src/screens/ — Screen components (WelcomeScreen, ValueSlidesScreen, PlaceholderScreen)
  • src/components/ — Reusable UI (Aurora, Button, ProgressDots, Screen)
  • src/theme/ — Design tokens (colors, type, motion, radii, layout)
  • src/state/ — Zustand stores (onboarding store with step, survey, briefingScore)
  • app/ — Expo Router routes (NOT YET CREATED — navigation targets defined in code)
  • __tests__/ — Snapshot & unit tests
  • .maestro/ — Maestro E2E flow tests
  • docs/ — Prototypes, specs, wireframes

1. Welcome BUILT
Screen:WelcomeScreen.tsx
Route:/onboarding/welcome
Content:Animated logo mark, "UPMIND" eyebrow, headline "Congratulations. You've taken the first step toward a sharper mind." Subtitle about 10-min daily practice. "Begin" CTA. "Already have an account? Sign in" link.
Nav:"Begin" → setStep(1) → router.push('/onboarding/value'). "Sign in" → router.push('/auth/sign-in').
Animation:Staggered sequence: logo (0ms), headline (120ms), subtitle (220ms), CTA (420ms), legal (560ms). Fade + translateY. Duration: 520ms. Ease: 0.22/1/0.36/1.
2. Value Slides BUILT
Screen:ValueSlidesScreen.tsx
Route:/onboarding/value
Slides:Slide 1 — Focus/Clarity: "Cut through noise. Decide faster." Concentric rings visual. Slide 2 — Memory: "Remember what actually matters." Network nodes visual. Slide 3 — Social Proof: "Used by founders, surgeons, and pilots." 4.9★, 18,400+ members, testimonial.
Nav:FlatList horizontal paging. "Continue" (slides 1-2) / "Take the survey" (slide 3). Final → setStep(2) → router.push('/onboarding/survey').
3. Survey STUB
Screen:PlaceholderScreen → will be SurveyScreen
Store:7 fields in Zustand: reason, struggle, priority, priming, commitment, age, occupation
Fields:reason (why here), struggle (biggest cognitive challenge), priority (what to improve), priming (experience with cognitive training), commitment (days/week), age, occupation.
4. Briefing STUB
Screen:PlaceholderScreen → will be BriefingScreen
Store:briefingScore (number | null) in onboarding store
Content:Summarizes survey responses, previews what to expect, shows initial cognitive estimate. "Start your journey" CTA leads to paywall.
5. Paywall STUB
Screen:PlaceholderScreen → will be PaywallScreen
Plans:Monthly and annual subscription tiers. Premium features: correlations, challenges, lifestyle logging, unlimited games.
6. Auth STUB
Screens:PlaceholderScreen → SignIn + SignUp screens
Routes:/auth/sign-in, /auth/sign-up
Flow:Email/password auth. Social sign-in options. "Already have an account?" / "Don't have an account?" links between sign-in and sign-up.
Future: Post-auth onboarding will include a 5-step philosophy carousel (intro → anti-gamification → lifestyle preview → privacy pledge → setup w/ email), then a 3-game baseline phase (Flanker, Digit Span, Mental Arithmetic) establishing reference scores before reaching the home dashboard.
